# JAVA

: 1995년 썬 마이크로시스템즈 (오라클에 인수) - 객체지향 언어
- C와 C++ 문법을 기본으로 차용 - 포인터, 다중상속 제거 -> 비교적 쉬운 언어
- 플랫폼에 독립적 - JVM() 만 있으면 윈도우, 리눅스, 맥등 어떤 플랫폼에서도 실행이 가능
- 자동 메모리 관리


## 개발순서
1. 소스 작성
2. 작성한 소스 컴파일
3. 컴파일한 소스를 JVM을 이용해 실행
